LangGraph Agent System
======================

Это мультиагентная система на основе LangGraph.

Доступные агенты:
1. RAG Agent - поиск информации по документам
2. SQL Agent - работа с базой данных
3. Chat Agent - диалоговый ассистент

Технологии:
- LangGraph для оркестрации
- FastAPI для REST API
- Kubernetes для развертывания
- Streamlit для веб-интерфейса
